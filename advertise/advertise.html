Formats de publicité possible

Vous pouvez choisir l'un des formats suivants. Pour plus d'exemples, merci de parcours les archives des précédentes éditions.

Format A

Publicité sous forme de text seul, s'intégrant parfaitement dans la mise en page des stories. Pas besoin d'images. Idéal pour promouvoir une entreprise ou un service.

Titre Principal 2–3 mots courts (nous pouvons le modifier)
Nom & Lien Nom de votre entreprise, produit ou service et le lien de destination
Résumé ~5–7 mots qui résument votre produit
Description La description votre produit (320 caractères max)

Format B

Une image avec un bloc de text. Ideal pour la véhiculer l'image de marque d'une entreprise, mise en avant de produits, annoncer un événement etc.

Titre Principal 3-4 mots courts (nous pouvons le modifier)
Description Le text principal (450 caractères max, un paragraphe seulement, peut contenir plusieurs liens)
Image Une image carrée (JPG seulement, 800 x 800 pixels). L'image ne doit pas être trop détaillée ou communiqué un message trop compliqué. Une simple photo, illustration ou graphique attire bien mieu l'attention.
